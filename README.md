# Iheb Belaid Portfolio

Portfolio personnel construit avec Astro, Tailwind CSS et DaisyUI.

## Prerequis

- Node.js 22+
- npm

## Installation

```bash
npm install
```

## Developpement local

```bash
npm run dev
```

## Build production

```bash
npm run build
```

## Preview local du build

```bash
npm run preview
```
